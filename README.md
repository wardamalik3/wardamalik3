## 👩‍💻 About Me

I'm an **entry-level SQA Engineer** and final-year **Software Engineering** student (CGPA **3.0**) at **FAST-NUCES**, obsessed with one question on every build: *"How can I break this on purpose, before a user breaks it by accident?"*

- 🧪 **5+ completed QA projects** spanning black-box, white-box, automation, and AI/RAG verification
- ☁️ **AWS Certified**, with hands-on cloud infrastructure exposure (EC2, VPC, RDS, IAM, Auto Scaling)
- 🧬 I think in **test coverage and edge cases** — equivalence classes, boundary values, decision points, and fault injection are part of how I read code, not just test it
- 🛠️ Comfortable owning the full QA lifecycle: test design → execution → defect logging in Jira → retesting → regression sign-off
- 🌐 Also build the things I test — full-stack MERN/Next.js apps — so I understand *both sides* of the bug report
- 📈 Currently deepening my CI/CD and DevOps skills to shift testing further left in the pipeline
